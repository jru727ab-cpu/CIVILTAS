Civilization-building strategy, now published live.

CIVILTAS

Lead a fledgling settlement from survival to Gnosis in this text-based Python game. Harvest resources, grow your population, raise monuments, and pursue enlightenment.

View the source on GitHub

What you do

  • Harvest food, wood, stone, and gold every turn.
  • Build farms, mines, temples, and libraries.
  • Balance growth, food consumption, and enlightenment.
  • Advance through four ages until you unlock Gnosis.

Play locally

Clone the repository and run the game with Python 3.8 or newer.

python civiltas.py
